## Coding Style

HTML tag contains two parts: head and body.
```html
<!DOCTYPE html>
<html>
	<head>

	</head>

	<body>

	</body>
</html>
```

The body tag contains 3 and must be 3 tags: header, main, footer

```html
<body>
	<header>

	</header>

	<main>

	</main>

	<footer>

	</footer>
</body>
```

The style of each part can be written inside each part.

```html
<header>
	.
	.
	.
	<style>

	</style>
</header>
```


```html
<main>

	.
	.
	.
	<style>

	</style>
</main>
```

```html
<footer>
	.
	.
	.
	<style>

	</style>
</footer>
```
